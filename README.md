# 이미지 파일 자체에 메모 남기는 프로그램 (picsay)

## 주요 기능 (Key Features)
* **속성>설명란에 메모:** 이미지 파일명으로는 쓰기 어렵거나 긴 내용을 파일 자체에 기록하기 위함
* **일괄 변경:** 일괄 수정 
* **일괄 저장:** 일괄 변경
