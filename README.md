# 🚀 이것저것 해보는 깃 (ColdBrew)

> **[혼자해보는 깃]** 개인 프로젝트 학습용 저장소입니다.
<br>

## 📌 주요 기능 (Key Features)
* **실시간 모니터링:** 5분마다 게시판을 스캔하여 새 글 개수 파악
* **푸시 알림:** 새로운 게시글 발견 시 윈도우 토스트(Toast) 팝업 알림 송출
* **로그 저장:** 예외 상황 발생 시 에러 로그를 `app.log` 파일로 자동 기록
<br>
